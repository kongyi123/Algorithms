

* 일반 적으로 전역변수는 프로젝트 내 모든 코드에서 공유된다. (extern 선언을 안했다 할지라도) // 초기화 구문이 없을때 자동적으로 extern 문구가 붙는다

* 한 파일내에서 전역 변수로 선언해서 쓰고 싶은데 다른 파일에서 참조하는 것은 막고싶은 경우 (전역 변수지만 extern이 아닌 경우) 는 전역변수에 static 문구를 붙이면 된다.
= static 변수를 만들면 extern이 안된다.
