사용자에게 디스크 일정량 할당
디스크의 부분을 쪼개어서 사용해도 됨
이것을 내가 임의로 할당 관리 해주는 프로그램

사용자에게 추가 할당도 가능
자원의 유효성은 디스크 배열이 있고 그것을 손대면 안됨

<입력 형식>

3

// 입력 타입(할당), 사용자ID, 크기
1 1 30

// 입력 타입(삭제), 사용자ID 
2 1

// 입력 타입(출력)
0 // 디스크 개수를 출력



* 단, 1~3 3~6 <- 이런식으로 범위라기 보다는 1~3 4~6 이런식으로 인덱스로 범위가 선택된다고 가정.

* 한 번 생성된 블록은 삭제되지 않는다.

* 남은 블록보다 큰 용량을 할당받아야 한다면 블록이 추가된다.
